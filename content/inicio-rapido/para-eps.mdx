# Inicio Rápido para EPS

Esta guía te ayudará a entender cómo el estándar Sofmed Interop impacta y beneficia las operaciones de tu EPS.

## ¿Qué cambia para tu EPS?

Con Sofmed Interop, tu EPS podrá:

1. **Recibir solicitudes de referencia** en formato estructurado FHIR
2. **Automatizar autorizaciones** basándose en datos estandarizados
3. **Hacer seguimiento en tiempo real** del estado de cada referencia
4. **Generar reportes** con datos consistentes y comparables

## Pasos para comenzar

### Paso 1: Entender el flujo
Revisa el [flujo completo de referencia](/docs/estandar/flujo-referencia) para entender qué datos se intercambian y en qué momentos.

### Paso 2: Revisar los perfiles FHIR relevantes
Los recursos FHIR que tu EPS necesita manejar son:
- [ServiceRequest](/docs/estandar/perfiles-fhir/service-request) — La solicitud de referencia
- [Patient](/docs/estandar/perfiles-fhir/patient) — Datos del paciente
- [Organization](/docs/estandar/perfiles-fhir/organization) — Datos de tu EPS y las IPS

### Paso 3: Evaluar tu sistema actual
Usa el [checklist de adopción para EPS](/docs/guia-eps/checklist-adopcion) para evaluar qué tan preparada está tu infraestructura.

### Paso 4: Coordinar con tu equipo técnico
Comparte la [guía para desarrolladores](/docs/guia-desarrolladores) con tu equipo de tecnología para iniciar la implementación.
