# Encounter

## Descripción

El perfil `SofmedEncounter` representa el encuentro clínico asociado a una referencia o contrareferencia. Registra cuándo, dónde y por qué se atendió al paciente.

**URL del perfil:** `https://sofmed.co/fhir/StructureDefinition/SofmedEncounter`

## Campos obligatorios

| Campo | Tipo | Descripción |
|-------|------|------------|
| `status` | `code` | Estado del encuentro (`planned`, `in-progress`, `finished`, `cancelled`) |
| `class` | `Coding` | Tipo de encuentro (`AMB`, `EMER`, `IMP`, `SS`) |
| `subject` | `Reference(Patient)` | Paciente |
| `period` | `Period` | Fecha/hora de inicio y fin |
| `serviceProvider` | `Reference(Organization)` | IPS que presta el servicio |

## Campos recomendados

| Campo | Tipo | Descripción |
|-------|------|------------|
| `participant` | `BackboneElement[]` | Profesionales involucrados |
| `diagnosis` | `BackboneElement[]` | Diagnósticos del encuentro |
| `reasonCode` | `CodeableConcept[]` | Motivo del encuentro |
| `serviceType` | `CodeableConcept` | Tipo de servicio (código CUPS) |

## Clases de encuentro

| Código | Descripción |
|--------|------------|
| `AMB` | Ambulatorio |
| `EMER` | Urgencias |
| `IMP` | Hospitalización |
| `SS` | Atención domiciliaria |

```json

  "resourceType": "Encounter",
  "meta": {
    "profile": ["https://sofmed.co/fhir/StructureDefinition/SofmedEncounter"]

    "system": "http://terminology.hl7.org/CodeSystem/v3-ActCode",
    "code": "AMB",
    "display": "ambulatory"
  },
  "subject": {
    "reference": "Patient/ejemplo-paciente"
  },
  "participant": [
    {
      "individual": {
        "reference": "Practitioner/ejemplo-medico"
      }

  "period": {
    "start": "2024-01-15T10:00:00-05:00",

  },

    {
      "condition": {
        "display": "Hipertensión esencial (primaria)"
      }
    }
  ],
  "serviceProvider": {
    "reference": "Organization/ejemplo-ips"
  }
}
```
